SUI-Label React component

An open source React card component

Description

The SUI-Label component features a generic SPAN tag including a centered themable custom literal.

SUI-Label render Example

SUI-Label Component implementation

    
  ReactDom.render(
        <Label literal={'Default Label'} />,
    document.getElementById('main')
  );
